# D & J Handmade Crafts

A showcase website for handcrafted crochet creations, featuring unique pieces made with traditional techniques and premium materials.

## About

D & J Handmade Crafts celebrates over 50 years of crafting experience, showcasing lovingly handcrafted crochet items including adorable animals like puppies, cats, turtles, unicorns, and foxes. Each piece reflects a commitment to quality, sustainability, and traditional craftsmanship techniques passed down through generations.

## Features

- **Craft Gallery**: Showcase of featured handcrafted items with images
- **Responsive Design**: Mobile-friendly interface for viewing on any device
- **About Section**: Information about the artisan and their crafting journey
- **Contact Section**: Easy way for customers to get in touch for custom orders

## Craft Collection

The current collection includes:
- Hand-crocheted puppies (sitting and standing)
- Adorable turtle pairs in various colors
- Charming cats with hearts
- Magical unicorns (sitting and standing)
- Warm and cozy foxes

Each piece is crafted by hand using time-honored techniques and quality thread.

## Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/yourusername/djhandmadecrafts.git
```

2. Navigate to the project directory:
```bash
cd djhandmadecrafts
```

3. Open `index.html` in your web browser or use a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js http-server
npx http-server
```

4. Visit `http://localhost:8000` in your browser
